📏 Правило образования сравнительной степени

Держите эту таблицу под рукой во время выполнения упражнений.

Односложные прилагательные+ ‑er (или + ‑r, если слово оканчивается на ‑e)
old → older nice → nicer
Одна гласная + одна согласная на концеудвоение согласной + ‑er
big → bigger hot → hotter
Оканчиваются на согласную + ‑y‑y меняется на ‑ier
dry → drier healthy → healthier
Односложные, оканчивающиеся на ‑edmore + прилагательное
tired → more tired
Двух- и более сложные прилагательныеmore + прилагательное
famous → more famous expensive → more expensive
Исключения (irregular)особые формы, не по правилам
good → better bad → worse far → further
