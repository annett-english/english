📎 Шпаргалка: когда gerund, а когда infinitive

Gerund (Verb + -ing)

После предлогов: interested in learning
В роли подлежащего: Reading is relaxing
После глаголов: enjoy, love, hate, mind, avoid, admit, deny, finish, keep, miss, practise, prefer, recommend, suggest, stop, give up

Infinitive (to + Verb)

После прилагательных: easy to learn
Чтобы показать цель: study to pass
После глаголов: want, need, hope, decide, learn, plan, promise, agree, expect, forget, refuse, offer, seem, afford

Infinitive без to

После modal verbs: can, must, should…
После make / let: let me stay, made me laugh

Verb + person + to + Verb

После ask, tell, want, would like: told me to wait

Значение меняется

try to do (постараться) / try doing (попробовать)
remember to do (не забыть) / remember doing (помнить, что делал)

Значение не меняется

start / begin / continue + to do = doing
like / love / hate + to do = doing
