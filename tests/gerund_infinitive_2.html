Gerund & Infinitive

Тренажёр по правилам: модальные глаголы, make/let, start/begin/continue, try, remember, like/love/hate

Правило 1 — Модальные глаголы (инфинитив без to)
После can, could, must, should, may, might, will, would глагол ставится в начальной форме без частицы to.
✔️ You must study.   ✔️ She can swim.
Правило 2 — make / let (инфинитив без to)
После make (заставлять) и let (позволять) в активном залоге глагол ставится без to.
✔️ My parents let me stay.   ✔️ She made me laugh.
Правило 3 — start / begin / continue (значение не меняется)
После этих глаголов можно использовать и gerund (-ing), и to-infinitive — смысл одинаковый.
✔️ It started to rain.   ✔️ It started raining.
Правило 4 — try (значение меняется)
try to do = постараться (приложить усилие)
try doing = попробовать ради эксперимента
✔️ Try to be on time.   ✔️ Try using this app.
Правило 5 — remember (значение меняется)
remember to do = не забыть сделать (сделать это ещё предстоит)
remember doing = помнить, что уже делал
✔️ Remember to call me.   ✔️ I remember meeting her.
Правило 6 — like / love / hate (значение не меняется)
Можно использовать и gerund, и to-infinitive.
✔️ I like reading.   ✔️ I like to read.
