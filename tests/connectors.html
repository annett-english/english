8 упражнений по 10 заданий. В каждом упражнении есть кнопки «Проверить» (подсветит верные/неверные ответы) и «Показать ответы» (покажет правильный вариант). Упражнение 8 — открытое: впишите свои идеи и скопируйте их для доски Miro.

01

Собери слово по буквам

Linking words. Впишите слово/фразу целиком — подсказка показывает первые буквы каждого слова.

02

Напиши слово

По русской подсказке напишите английское связующее слово (linking word).

03

Rephrase

Перепишите предложение так, чтобы в нём использовалось указанное слово (смысл должен сохраниться). Проверка мягкая: важно, чтобы нужное слово было в вашем ответе — итоговый вариант сверьте с образцом.

04

Контекст

Выберите слово, которое подходит по смыслу.

05

Собери словосочетание

Verb + phrase collocations. Выберите из списка фразу, которая сочетается с глаголом.

06

Собери слово по буквам

Вставьте пропущенный глагол из словосочетания в предложение.

07

Контекст

Выберите подходящий глагол для словосочетания.

08

Закончи предложение

Открытое задание — единственно верного ответа нет. Допишите предложения своими идеями, затем скопируйте текст и вставьте его на доску Miro.
