Grammar workout · agreement & short additions

“I love grammar.” — So do I!

Practise agreeing with people: too / either and So do I / Neither do I — in different tenses and with modal verbs.

Quick reference — how it works

✓ Agreeing with positive

“I like coffee.” → “I like it too.” / “Me too.”

“I like coffee.” → “So do I.”

“She can swim.” → “So can I.”

“He has been to Rome.” → “So have I.”

So + auxiliary + subject (inverted word order!)

✗ Agreeing with negative

“I don't like tea.” → “I don't like it either.” / “Me neither.”

“I don't like tea.” → “Neither do I.” / “Nor do I.”

“She can't drive.” → “Neither can I.”

“He wasn't there.” → “Neither was I.”

Neither/Nor + auxiliary + subject — no “not”! (Neither is already negative.)
Which auxiliary? Copy the “engine” of the first sentence: Present Simple → do / does · Past Simple → did · Continuous → am / is / are / was / were · Perfect → have / has / had · Future → will · Modals → can, could, must, should, would, may…
