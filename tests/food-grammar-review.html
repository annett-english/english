1

a / an — «один, какой-то»

Ставим только перед исчисляемым существительным в единственном числе. Выбор зависит от звука, а не буквы.

a

перед согласным звуком

a pear — (одна) груша

a pepper — (один) перец

a strawberry — (одна) клубничина

an

перед гласным звуком

an onion — (одна) луковица

an apple — (одно) яблоко

an egg — (одно) яйцо

Нельзя: a salmon (о рыбе-еде), an asparagus, a cabbage soup* — с неисчисляемыми артикль a/an не ставится. Говорим: salmon, some asparagus.
* «a cabbage» возможно, когда имеем в виду один кочан как предмет: We bought a cabbage.
2

Две корзины: считаем или нет?

Исчисляемое — можно посчитать по одному (one pear, two pears). Неисчисляемое — масса, вещество: нет множественного числа и нет a/an.

🧺 Countable исчисляемые

one … / two …s · можно a/an

  • a pear → pearsгруша
  • an onion → onionsлуковица
  • a strawberry → strawberriesклубничина
  • a pineapple → pineapplesананас (целый)
  • a pancake → pancakesсырник / блинчик
  • a pepper → peppersперец (болгарский, штука)

🥣 Uncountable неисчисляемые

без a/an, без -s · глагол в ед. числе

  • pepperперец (специя)
  • beetrootсвёкла
  • cabbageкапуста (как продукт)
  • asparagusспаржа
  • salmonлосось (как еда)
  • pineappleананас (нарезанный, как продукт)
Хитрые слова: pepper, pineapple, cabbage бывают в обеих корзинах — целый предмет (штука) → исчисляемое, вещество/еда на тарелке → неисчисляемое. I ate pineapple (ел ананас как еду) vs I bought a pineapple (купил один целый ананас).
3

some / any — «немного, несколько»

Работают и с неисчисляемыми (some salmon), и с исчисляемыми во множественном числе (some strawberries).

ГдеСловоПример
Утверждение ( + ) some We have some beetroot.У нас есть (немного) свёклы.
Вопрос ( ? ) any Do we have any onions?У нас есть лук?
Отрицание ( − ) any There isn't any salmon left.Лосося не осталось.
Исключение: предложение / просьба some Would you like some pancakes? · Can I have some pepper?Хотите сырников? · Можно мне перца?
