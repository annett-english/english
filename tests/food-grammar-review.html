1

a / an — «один, какой-то»

Ставим перед существительным в единственном числе. Выбор зависит от звука, а не буквы.

a

перед согласным звуком

a pear — (одна) груша

a pepper — (один) болгарский перец

a strawberry — (одна) клубничина

an

перед гласным звуком

an onion — (одна) луковица

an apple — (одно) яблоко

an egg — (одно) яйцо

Без a/an: salmon, asparagus, beetroot, cabbage — их не считаем по штукам. Говорим просто salmon или some salmon, но не a salmon.
2

some / any — «немного, несколько»

Работают со словами вроде salmon, beetroot (some salmon) и со множественным числом (some strawberries).

ГдеСловоПример
Утверждение ( + ) some We have some beetroot.У нас есть (немного) свёклы.
Вопрос ( ? ) any Do we have any onions?У нас есть лук?
Отрицание ( − ) any There isn't any salmon left.Лосося не осталось.
